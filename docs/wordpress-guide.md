# Wordpress - Starting guide

## What is wordpress?
WordPress is a CMS (Content Management System) written in PHP. It allows users to 
create, manage, and serve dynamic websites without writing code. 
WordPress runs on the server, processing each request and 
generating pages on the fly by pulling content from a database.
